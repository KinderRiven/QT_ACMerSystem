mysqlServer/
	comPort.exe 串口监视程序
	mysqlServer.jar java实现的数据库连接文件
	run.bat 调用.jar文件的bat可执行文件
	config 数据库连接设置 包括（数据库名称，IP,端口，数据存储地址，刷卡用户记录文件名，刷卡用户时间记录文件名，数据库账号，密码）
	pconfgi python端口监视程序设置文件 （端口号，bat文件，数据库查找表名字，数据库索引列名字）
	time 刷卡时间记录文件
	user 刷卡用户记录文件
	data/ 用户信息临时记录文件

code/ 源代码文件夹（python、java程序）
Resources/ 资源文件（图片，声音）
